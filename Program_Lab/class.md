2. 클래스 ```Date```와 관련한 멤버함수를 구현하시오([참조](../SampleCodes/Class/dateClass.cc)).

```C++
class Date{
	int year;
	int month;
	int day;
public:
	Date()
	Date(int y, int m, int d);
	~Date();
	void setYear(int y);
	void setMonth(int m);
	void setDay(int d);
	int getYear();
	int getMonth();
	int getDay();
	void printDate();
};


```
